*javax.management.openmbean.OpenMBeanParameterInfo_Description*

Describes a parameter used in one or more operations or constructors of an open 
MBean. 

This interface declares the same methods as the class 
(|javax.management.MBeanParameterInfo|) . A class implementing this interface 
(typically (|javax.management.openmbean.OpenMBeanParameterInfoSupport|) )
