*org.omg.CORBA.IDLType_Description*

An abstract interface inherited by all Interface Repository (IR) objects that 
represent OMG IDL types. It provides access to the TypeCode object describing 
the type and is used in defining the other interfaces wherever definitions of 
IDLType must be referenced.
