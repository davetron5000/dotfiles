*javax.rmi.CORBA.PortableRemoteObjectDelegate_Description*

Supports delegation for method implementations in (|javax.rmi.PortableRemoteObject|) . The delegate is a singleton instance of a class that implements this interface and provides 
a replacement implementation for all the methods of javax.rmi.PortableRemoteObject. 

Delegates are enabled by providing the delegate's class name as the value of the javax.rmi.CORBA.PortableRemoteObjectClass system property.
