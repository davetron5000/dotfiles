*java.net.ConnectException_Description*

Signals that an error occurred while attempting to connect a socket to a remote address and port. Typically, the connection was refused remotely (e.g., no process is listening on 
the remote address/port). 



*java.net.ConnectException()*

public ConnectException()

Construct a new ConnectException with no detailed message. 


*java.net.ConnectException(String)*

public ConnectException(java.lang.String msg)

Constructs a new ConnectException with the specified detail message as to why the connect error occurred. A detail message is a String that gives a specific description of this 
error. 

    msg - the detail message
