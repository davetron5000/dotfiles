*org.w3c.dom.html.HTMLIsIndexElement_Description*

This element is used for single-line text input. See the ISINDEX element 
definition in HTML 4.0. This element is deprecated in HTML 4.0. See also the 
Document Object Model (DOM) Level 2 Specification.
