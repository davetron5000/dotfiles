*javax.sound.midi.MetaEventListener_Description*

The MetaEventListener interface should be implemented by classes whose 
instances need to be notified when a (|javax.sound.midi.Sequencer|) has 
processed a (|javax.sound.midi.MetaMessage|) . To register a MetaEventListener 
object to receive such notifications, pass it as the argument to the 
addMetaEventListener(|javax.sound.midi.Sequencer|) method of Sequencer.
