*javax.management.AttributeChangeNotificationFilter_Description*

This class implements of the 
NotificationFilter(|javax.management.NotificationFilter|) interface for the 
attribute change notification(|javax.management.AttributeChangeNotification|) . 
The filtering is performed on the name of the observed attribute. 

It manages a list of enabled attribute names. A method allows users to 
enable/disable as many attribute names as required.

*javax.management.AttributeChangeNotificationFilter.isNotificationEnabled(Notification)*

public synchronized boolean isNotificationEnabled(javax.management.Notification notification)

Invoked before sending the specified notification to the listener. This filter 
compares the attribute name of the specified attribute change notification with 
each enabled attribute name. If the attribute name equals one of the enabled 
attribute names, the notification must be sent to the listener and this method 
returns true. 

    notification - The attribute change notification to be sent. 

    Returns: true if the notification has to be sent to the listener, false otherwise.
