*javax.swing.text.StyledEditorKit.ForegroundAction_Description*

An action to set foreground color. This sets the StyleConstants.Foreground attribute for the currently selected range of the target JEditorPane. This is done by calling 
StyledDocument.setCharacterAttributes on the styled document associated with the target JEditorPane. 

If the target text component is specified as the source of the ActionEvent and there is a command string, the command string will be interpreted as the foreground color. It will 
be interpreted by called Color.decode, and should therefore be legal input for that method.
