*javax.management.QueryExp_Description*

Represents relational constraints that can be used in database query "where 
clauses". Instances of QueryExp are returned by the static methods of the 
(|javax.management.Query|) class. 

It is possible, but not recommended, to create custom queries by implementing 
this interface. In that case, it is better to extend the 
(|javax.management.QueryEval|) class than to implement the interface directly, 
so that the (|javax.management.QueryExp|) method works correctly.
