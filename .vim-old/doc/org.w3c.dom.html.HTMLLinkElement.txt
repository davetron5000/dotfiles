*org.w3c.dom.html.HTMLLinkElement_Description*

The LINK element specifies a link to an external resource, and defines this 
document's relationship to that resource (or vice versa). See the LINK element 
definition in HTML 4.0 (see also the LinkStyle interface in the module). See 
also the Document Object Model (DOM) Level 2 Specification.

public boolean getDisabled()

Enables/disables the link. This is currently only used for style sheet links, 
and may be used to activate or deactivate style sheets.
