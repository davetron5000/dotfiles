*java.util.FormattableFlags_Description*

FomattableFlags are passed to the Formattable.formatTo()(|java.util.Formattable|) method and modify the output format for Formattables(|java.util.Formattable|) . Implementations 
of (|java.util.Formattable|) are responsible for interpreting and validating any flags. 



*int_java.util.FormattableFlags.ALTERNATE*

Requires the output to use an alternate form. The definition of the form is specified by the Formattable. 

This flag corresponds to '#' ('u0023') in the format specifier. 


*int_java.util.FormattableFlags.LEFT_JUSTIFY*

Left-justifies the output. Spaces ('u0020') will be added at the end of the converted value as required to fill the minimum width of the field. If this flag is not set then the 
output will be right-justified. 

This flag corresponds to '-' ('u002d') in the format specifier. 


*int_java.util.FormattableFlags.UPPERCASE*

Converts the output to upper case according to the rules of the locale(|java.util.Locale|) given during creation of the formatter argument of the 
formatTo()(|java.util.Formattable|) method. The output should be equivalent the following invocation of (|java.lang.String|) 

out.toUpperCase() 

This flag corresponds to '^' ('u005e') in the format specifier.
