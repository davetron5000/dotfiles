*javax.naming.ldap.UnsolicitedNotification_Description*

This interface represents an unsolicited notification as defined in RFC 2251. An unsolicited notification is sent by the LDAP server to the LDAP client without any provocation 
from the client. Its format is that of an extended response (ExtendedResponse).
