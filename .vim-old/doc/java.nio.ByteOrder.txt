*java.nio.ByteOrder_Description*

A typesafe enumeration for byte orders. 



*java.nio.ByteOrder_java.nio.ByteOrder.BIG_ENDIAN*

Constant denoting big-endian byte order. In this order, the bytes of a multibyte value are ordered from most significant to least significant. 


*java.nio.ByteOrder_java.nio.ByteOrder.LITTLE_ENDIAN*

Constant denoting little-endian byte order. In this order, the bytes of a multibyte value are ordered from least significant to most significant. 



*java.nio.ByteOrder.nativeOrder()*

public static |java.nio.ByteOrder| nativeOrder()

Retrieves the native byte order of the underlying platform. 

This method is defined so that performance-sensitive Java code can allocate direct buffers with the same byte order as the hardware. Native code libraries are often more efficient 
when such buffers are used. 



    Returns: The native byte order of the hardware upon which this Java virtual machine is running
