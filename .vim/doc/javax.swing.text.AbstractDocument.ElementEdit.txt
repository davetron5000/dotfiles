  javax.swing.text.Element[] removed,
  javax.swing.text.Element[] added)

Constructs an edit record. This does not modify the element so it can safely be 
used to catch up a view to the current model state for views that just attached 
to a model. 

    e - the element 
    index - the index into the model >= 0 
    removed - a set of elements that were removed 
    added - a set of elements that were added
