*java.awt.RenderingHints.Key_Description*

Defines the base type of all keys used to control various aspects of the 
rendering and imaging pipelines. Instances of this class are immutable and 
unique which means that tests for matches can be made using the == operator 
instead of the more expensive equals() method. 


*java.awt.RenderingHints.Key(int)*

protected RenderingHints.Key(int privatekey)

Construct a key using the indicated private key. Each subclass of Key maintains 
its own unique domain of integer keys. No two objects with the same integer key 
and of the same specific subclass can be constructed. An exception will be 
thrown if an attempt is made to construct another object of a given class with 
the same integer key as a pre-existing instance of that subclass of Key. 

    privatekey - the specified key
