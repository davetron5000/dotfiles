*java.awt.print.PrinterGraphics_Description*

The PrinterGraphics interface is implemented by (|java.awt.Graphics|) objects 
that are passed to (|java.awt.print.Printable|) objects to render a page. It 
allows an application to find the (|java.awt.print.PrinterJob|) object that is 
controlling the printing.
