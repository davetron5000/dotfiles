*org.w3c.dom.html.HTMLBodyElement_Description*

The HTML document body. This element is always present in the DOM API, even if 
the tags are not present in the source document. See the BODY element 
definition in HTML 4.0. See also the Document Object Model (DOM) Level 2 
Specification.
