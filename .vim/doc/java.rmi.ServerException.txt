*java.rmi.ServerException_Description*

A ServerException is thrown as a result of a remote method invocation when a 
RemoteException is thrown while processing the invocation on the server, either 
while unmarshalling the arguments or executing the remote method itself. 

A ServerException instance contains the original RemoteException that occurred 
as its cause.
