*javax.print.attribute.HashDocAttributeSet(DocAttribute[])*

public HashDocAttributeSet(javax.print.attribute.DocAttribute[] attributes)

Construct a new hash doc attribute set, initially populated with the values 
from the given array. The new attribute set is populated by adding the elements 
of attributes array to the set in sequence, starting at index 0. Thus, later 
array elements may replace earlier array elements if the array contains 
duplicate attribute values or attribute categories. 

    attributes - Array of attribute values to add to the set. If null, an empty attribute set is 
       constructed.
