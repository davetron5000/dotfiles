*javax.print.URIException_Description*

Interface URIException is a mixin interface which a subclass of 
PrintException(|javax.print.PrintException|) can implement to report an error 
condition involving a URI address. The Print Service API does not define any 
print exception classes that implement interface URIException, that being left 
to the Print Service implementor's discretion.
