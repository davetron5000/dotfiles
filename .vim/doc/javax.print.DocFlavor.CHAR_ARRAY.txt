*javax.print.DocFlavor.CHAR_ARRAY_Description*

Class DocFlavor.CHAR_ARRAY provides predefined static constant DocFlavor 
objects for example doc flavors using a character array (char[]) as the print 
data representation class. As such, the character set is Unicode.
