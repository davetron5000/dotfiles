*javax.swing.text.StyleContext.SmallAttributeSet_Description*

This class holds a small number of attributes in an array. The storage format 
is key, value, key, value, etc. The size of the set is the length of the array 
divided by two. By default, this is the class that will be used to store 
attributes when held in the compact sharable form.
