*javax.swing.text.html.HTMLDocument.HTMLReader.TagAction_Description*

An action to be performed in response to parsing a tag. This allows 
customization of how each tag is handled and avoids a large switch statement.

public void end(javax.swing.text.html.HTML.Tag t)

Called when an end tag is seen for the type of tag this action was registered 
to. The tag argument indicates the actual tag for those actions that are shared 
across many tags. By default this does nothing and completely ignores the tag.
