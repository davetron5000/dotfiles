*java.security.spec.X509EncodedKeySpec_Description*

This class represents the ASN.1 encoding of a public key, encoded according to 
the ASN.1 type SubjectPublicKeyInfo. The SubjectPublicKeyInfo syntax is defined 
in the X.509 standard as follows: 



SubjectPublicKeyInfo ::= SEQUENCE { algorithm AlgorithmIdentifier, 
subjectPublicKey BIT STRING }

       contents of the array are copied to protect against subsequent 
       modification.
