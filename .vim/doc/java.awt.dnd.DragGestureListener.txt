*java.awt.dnd.DragGestureListener_Description*

This interface is sourced from a DragGestureRecognizer and is invoked when an 
object of that (sub)class detects a drag initiating gesture. 

The implementor of this interface is responsible for starting the drag as a 
result of receiving such notification.
