*java.awt.PopupMenu.AccessibleAWTPopupMenu_Description*

Inner class of PopupMenu used to provide default support for accessibility. This class is not meant to be used directly by application developers, but is instead meant only to be 
subclassed by menu component developers. 

The class used to obtain the accessible role for this object.
