*javax.print.attribute.standard.DateTimeAtProcessing_Description*

Class DateTimeAtProcessing is a printing attribute class, a date-time attribute, that indicates the date and time at which the Print Job first began processing. 

To construct a DateTimeAtProcessing attribute from separate values of the year, month, day, hour, minute, and so on, use a Calendar(|java.util.Calendar|) object to construct a 
Date(|java.util.Date|) object, then use the Date(|java.util.Date|) object to construct the DateTimeAtProcessing attribute. To convert a DateTimeAtProcessing attribute to separate 
values of the year, month, day, hour, minute, and so on, create a Calendar(|java.util.Calendar|) object and set it to the Date(|java.util.Date|) from the DateTimeAtProcessing 
attribute. 

IPP Compatibility: The information needed to construct an IPP "date-time-at-processing" attribute can be obtained as described above. The category name returned by getName() gives 
the IPP attribute name.
