*java.util.concurrent.atomic.AtomicBoolean_Description*

A boolean value that may be updated atomically. See the 
(|java.util.concurrent.atomic|) package specification for description of the 
properties of atomic variables. An AtomicBoolean is used in applications such 
as atomically updated flags, and cannot be used as a replacement for a 
(|java.lang.Boolean|) .
