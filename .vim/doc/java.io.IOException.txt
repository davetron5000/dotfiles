*java.io.IOException_Description*

Signals that an I/O exception of some sort has occurred. This class is the 
general class of exceptions produced by failed or interrupted I/O operations. 


*java.io.IOException()*

public IOException()

Constructs an IOException with null as its error detail message. 


*java.io.IOException(String)*

public IOException(java.lang.String s)

Constructs an IOException with the specified detail message. The error message 
string s can later be retrieved by the (|java.lang.Throwable|) method of class 
java.lang.Throwable. 

    s - the detail message.
