*java.lang.Character.Subset_Description*

Instances of this class represent particular subsets of the Unicode character 
set. The only family of subsets defined in the Character class is 
UnicodeBlock(|java.lang.Character.UnicodeBlock|) . Other portions of the Java 
API may define other subsets for their own purposes.
