*javax.xml.datatype.XMLGregorianCalendar_Description*

Representation for W3C XML Schema 1.0 date/time datatypes. Specifically, these 
date/time datatypes are dateTime, time, date, gYearMonth, gMonthDay, gYear 
gMonth and gDay defined in the XML Namespace 
"http://www.w3.org/2001/XMLSchema". These datatypes are normatively defined in 
W3C XML Schema 1.0 Part 2, Section 3.2.7-14. 

The table below defines the mapping between XML Schema 1.0 date/time datatype 
fields and this class' fields. It also summarizes the value constraints for the 
date and time fields defined in W3C XML Schema 1.0 Part 2, Appendix D, ISO 8601 
Date and Time Formats.

custom timezone id using defaults.getTimezone(). else use the GregorianCalendar 
default timezone value for the host is defined as specified by 
java.util.TimeZone.getDefault().

should be instantiated in following manner.
