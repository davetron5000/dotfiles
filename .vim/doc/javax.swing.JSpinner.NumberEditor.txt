*javax.swing.JSpinner.NumberEditor_Description*

An editor for a JSpinner whose model is a SpinnerNumberModel. The value of the 
editor is displayed with a JFormattedTextField whose format is defined by a 
NumberFormatter instance whose minimum and maximum properties are mapped to the 
SpinnerNumberModel. 


*javax.swing.JSpinner.NumberEditor(JSpinner)*

public JSpinner.NumberEditor(javax.swing.JSpinner spinner)

Construct a JSpinner editor that supports displaying and editing the value of a 
SpinnerNumberModel with a JFormattedTextField. This NumberEditor becomes both a 
ChangeListener on the spinner and a PropertyChangeListener on the new 
JFormattedTextField. 

    spinner - the spinner whose model this editor will monitor
