*javax.swing.AbstractSpinnerModel_Description*

This class provides the ChangeListener part of the SpinnerModel interface that 
should be suitable for most concrete SpinnerModel implementations. Subclasses 
must provide an implementation of the minimum, maximum, and value properties 
and the getNextValue and getPreviousValue methods.
