*java.security.spec.ECField_Description*

This interface represents an elliptic curve (EC) finite field. All specialized 
EC fields must implements this interface. 


*java.security.spec.ECField.getFieldSize()*

public int getFieldSize()

Returns the field size in bits. Note: For prime finite field ECFieldFp, size of 
prime p in bits is returned. For characteristic 2 finite field ECFieldF2m, m is 
returned. 


    Returns: the field size in bits.
