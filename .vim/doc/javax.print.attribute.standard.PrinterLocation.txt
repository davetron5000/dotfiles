*javax.print.attribute.standard.PrinterLocation_Description*

Class PrinterLocation is a printing attribute class, a text attribute, that 
identifies the location of the device. This could include things like: "in Room 
123A, second floor of building XYZ". 

IPP Compatibility: The string value gives the IPP name value. The locale gives 
the IPP natural language. The category name returned by getName() gives the IPP 
attribute name.
