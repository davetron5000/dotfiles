*java.security.ProviderException_Description*

A runtime exception for Provider exceptions (such as misconfiguration errors or 
unrecoverable internal errors), which may be subclassed by Providers to throw 
specialized, provider-specific runtime errors. 


*java.security.ProviderException()*

public ProviderException()

Constructs a ProviderException with no detail message. A detail message is a 
String that describes this particular exception.

public ProviderException(java.lang.Throwable cause)

Creates a ProviderException with the specified cause and a detail message of
