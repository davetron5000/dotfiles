*javax.print.event.PrintJobAttributeEvent_Description*

Class PrintJobAttributeEvent encapsulates an event a PrintService reports to 
let the client know that one or more printing attributes for a PrintJob have 
changed. 


*javax.print.event.PrintJobAttributeEvent(DocPrintJob,PrintJobAttributeSet)*
