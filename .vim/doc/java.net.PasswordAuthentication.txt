*java.net.PasswordAuthentication_Description*

The class PasswordAuthentication is a data holder that is used by 
Authenticator. It is simply a repository for a user name and a password. 


*java.net.PasswordAuthentication(String,char[])*

public PasswordAuthentication(
  java.lang.String userName,
  char[] password)

Creates a new PasswordAuthentication object from the given user name and 
password. 

Note that the given user password is cloned before it is stored in the new 
PasswordAuthentication object. 

    userName - the user name 
    password - the user's password
