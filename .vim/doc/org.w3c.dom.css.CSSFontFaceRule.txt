*org.w3c.dom.css.CSSFontFaceRule_Description*

The CSSFontFaceRule interface represents a @font-face rule in a CSS style 
sheet. The @font-face rule is used to hold a set of font descriptions. See also 
the Document Object Model (DOM) Level 2 Style Specification.
