*java.security.KeyPair_Description*

This class is a simple holder for a key pair (a public key and a private key). 
It does not enforce any security, and, when initialized, should be treated like 
a PrivateKey. 


*java.security.KeyPair(PublicKey,PrivateKey)*

public KeyPair(
  java.security.PublicKey publicKey,
  java.security.PrivateKey privateKey)

Constructs a key pair from the given public key and private key. 

Note that this constructor only stores references to the public and private key 
components in the generated key pair. This is safe, because Key objects are 
immutable. 

    publicKey - the public key. 
    privateKey - the private key.
