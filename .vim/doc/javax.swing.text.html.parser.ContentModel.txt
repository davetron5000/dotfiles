*javax.swing.text.html.parser.ContentModel_Description*

A representation of a content model. A content model is basically a restricted BNF expression. It is restricted in the sense that it must be deterministic. This means that you 
don't have to represent it as a finite state automata. See Annex H on page 556 of the SGML handbook for more information.
