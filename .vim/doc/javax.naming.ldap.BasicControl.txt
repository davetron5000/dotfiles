*javax.naming.ldap.BasicControl.getEncodedValue()*

public byte getEncodedValue()

Retrieves the control's ASN.1 BER encoded value. The result includes the BER 
tag and length for the control's value but does not include the control's 
object identifier and criticality setting. 


    Returns: A possibly null byte array representing the control's ASN.1 BER encoded value. 
             It is not cloned - any changes to the returned value will affect 
             the contents of the control.
