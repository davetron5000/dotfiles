*javax.swing.plaf.basic.BasicMenuItemUI.doClick(MenuSelectionManager)*

protected void doClick(javax.swing.MenuSelectionManager msm)

Call this method when a menu item is to be activated. This method handles some 
of the details of menu item activation such as clearing the selected path and 
messaging the JMenuItem's doClick() method. 

    msm - A MenuSelectionManager. The visual feedback and internal bookkeeping tasks are 
       delegated to this MenuSelectionManager. If null is passed as this 
       argument, the MenuSelectionManager.defaultManager is used.
