*javax.xml.transform.dom.DOMLocator_Description*

Indicates the position of a node in a source DOM, intended primarily for error 
reporting. To use a DOMLocator, the receiver of an error must downcast the 
(|javax.xml.transform.SourceLocator|) object returned by an exception. A 
(|javax.xml.transform.Transformer|) may use this object for purposes other than 
error reporting, for instance, to indicate the source node that originated a 
result node.
