*javax.security.auth.x500.X500PrivateCredential_Description*

This class represents an X500PrivateCredential. It associates an X.509 
certificate, corresponding private key and the KeyStore alias used to reference 
that exact key pair in the KeyStore. This enables looking up the private 
credentials for an X.500 principal in a subject.
