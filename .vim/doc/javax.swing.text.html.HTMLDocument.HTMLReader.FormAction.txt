*javax.swing.text.html.HTMLDocument.HTMLReader.FormAction_Description*

Action to support forms by building all of the elements used to represent form 
controls. This will process the <INPUT>, <TEXTAREA>, <SELECT>, and <OPTION> 
tags. The element created by this action is expected to have the attribute 
StyleConstants.ModelAttribute set to the model that holds the state for the 
form control. This enables multiple views, and allows document to be iterated 
over picking up the data of the form. The following are the model assignments 
for the various type of form elements.

input, type button (|javax.swing.DefaultButtonModel|) input, type checkbox 
(|javax.swing.JToggleButton.ToggleButtonModel|) input, type image 
(|javax.swing.DefaultButtonModel|) input, type password 
(|javax.swing.text.PlainDocument|) input, type radio 
(|javax.swing.JToggleButton.ToggleButtonModel|) input, type reset 
(|javax.swing.DefaultButtonModel|) input, type submit 
(|javax.swing.DefaultButtonModel|) input, type text or type is null. 
(|javax.swing.text.PlainDocument|) select (|javax.swing.DefaultComboBoxModel|) 
or an (|javax.swing.DefaultListModel|) , with an item type of Option
