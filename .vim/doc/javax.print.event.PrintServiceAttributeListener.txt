*javax.print.event.PrintServiceAttributeListener_Description*

Implementations of this listener interface are attached to a 
PrintService(|javax.print.PrintService|) to monitor the status of the print 
service. 

To monitor a particular job see (|javax.print.event.PrintJobListener|) and 
(|javax.print.event.PrintJobAttributeListener|) . 


*javax.print.event.PrintServiceAttributeListener.attributeUpdate(PrintServiceAttributeEvent)*

public void attributeUpdate(javax.print.event.PrintServiceAttributeEvent psae)

Called to notify a listener of an event in the print service. The service will 
call this method on an event notification thread. The client should not perform 
lengthy processing in this callback or subsequent event notifications may be 
blocked. 

    psae - the event being notified
