*javax.print.event.PrintJobAttributeListener_Description*

Implementations of this interface are attached to a 
DocPrintJob(|javax.print.DocPrintJob|) to monitor the status of attribute 
changes associated with the print job. 


*javax.print.event.PrintJobAttributeListener.attributeUpdate(PrintJobAttributeEvent)*

public void attributeUpdate(javax.print.event.PrintJobAttributeEvent pjae)

Notifies the listener of a change in some print job attributes. One example of 
an occurrence triggering this event is if the 
JobState(|javax.print.attribute.standard.JobState|) attribute changed from 
PROCESSING to PROCESSING_STOPPED. 

    pjae - the event.
